## How to Use

When you notice something about the chassis that could be better while working on this project — a missing sub-task, a prompt that didn't work well, a handover field you wish existed — note it here. Don't stop your project work to fix the template. Capture it and move on.

Review this file weekly or between phases. Apply improvements to the template repo.
